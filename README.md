# 🚀 Selenium TestNG Automation Framework

## 📌 Project Description
This project is a test automation framework built using:
- Java
- Selenium WebDriver
- TestNG
- Maven

It follows Page Object Model (POM) design pattern and includes:
- Retry mechanism
- Screenshot on failure
- Clean BasePage structure

---

## 🧱 Tech Stack
- Java 17
- Selenium 4
- TestNG
- Maven

---

## 📂 Project Structure
- base → test setup
- pages → page objects
- tests → test cases
- utils → helper classes

---

▶️ How to Run Tests
mvn clean test or run via IntelliJ (right click → Run)
---
📸 Features

✅ Page Object Model (POM)
🔁 Retry failed tests
📸 Screenshot on failure
⚙️ Maven build support
